# Birthday Website

This project is a birthday website created to celebrate my boyfriend's special day. The website features a visually appealing design with interactive elements that allow users to explore heartfelt messages.

## Project Structure

- **index.html**: The main HTML document that contains the structure of the webpage, including the vase image and a container for the notecards.
- **styles.css**: Contains the styles for the website, setting the background color to baby blue and formatting the vase and notecards.
- **script.js**: Handles the interactivity of the website, including displaying notecards and their messages.

## Features

- A clear vase filled with white notecards.
- Clicking on the vase reveals an array of notecards.
- Each notecard can be clicked to display a personalized message.

## Setup Instructions

1. Clone the repository to your local machine.
2. Open the `index.html` file in a web browser to view the website.
3. Ensure that all files (`styles.css` and `script.js`) are in the same directory as `index.html` for proper functionality.

## Usage

- Click on the vase to view the notecards.
- Click on each notecard to read the special messages.

Enjoy exploring the birthday website!
